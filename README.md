# ArchCord
ArchCord is a custom discord bot that add a full linux file system to discord.
To add new commands, add them in data/SERVER/commands
to add a new module, like OS, or NET. add it in data/SERVER
makesure to add your token,and your ID into bot.py at the top.

Use database manager to get simpler access to users files, which makes it so you can only view files.

#### Note : there will be a README in the data folder with exact instructions on how to install and a list of bugs
