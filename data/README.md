## To use :

### 1 - Clone or Unzip to a folder

### 2 - Get your discord user ID :
####    2A - enable Developer mode in Discord settings
####    2B - right click a message you have send / your profile in the members list
####    2C - click "Copy user ID"
####    2D - delete idhere in bot.py line 21
####    2E - paste your user ID where idhere was

### 3 - Create a Discord bot
####   3A - go to the discord developer portal
####    3B - create a bot, fill in the needed fields
####    3C - Copy bot token (if you dont copy it before reloading the page the token will need to be reset)
####    3D - delete TokenHERE in bot.py line 24
####    3E - paste bot token where TokenHERE was

### 4 - Run the bot
####    4A - cd into the folder with bot.py
####    4B - run "python3 bot.py"
